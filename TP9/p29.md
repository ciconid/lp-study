## Pregunta 29
> Defina en Haskell de manera curried la forma funcional filter, la cual aplica un predicado a los elementos de una lista y retorna la lista formada por los elementos que satisfacen el predicado. Por ejemplo:
> ```haskell
> filter par [1,3,6,10,15,20] -- [6,10,20]
> ```

### Respuesta

La función `filter` en Haskell es una forma funcional curried, ya que toma un predicado y una lista, y retorna una nueva lista con los elementos que cumplen el predicado.

- **Tipo:** `(a -> Bool) -> [a] -> [a]`
- **Polimórfica:** Sí
- **Forma funcional:** Sí

```haskell
filter _ [] = []
filter p (x:xs)
  | p x       = x : filter p xs
  | otherwise = filter p xs
```

> *"Una forma funcional es una función que toma otra función como argumento o devuelve una función como resultado. La función filter es un ejemplo clásico de forma funcional en Haskell."*
> — Slides - Paradigma Funcional - Haskell y SML (p. 35)
