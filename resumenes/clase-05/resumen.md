# Resumen Clase 5: Tipos

## Introducción
Los tipos son fundamentales en los lenguajes de programación, ya que definen el conjunto de valores y operaciones permitidas para los datos.

## Tipos básicos o primitivos
- Definidos en el diseño del lenguaje (números, caracteres, booleanos, etc.)

## Tipos compuestos
- Permiten combinar tipos simples en estructuras más complejas (arreglos, registros, conjuntos, subrangos, punteros, archivos, etc.)

## Tipos definidos por el usuario
- El programador puede crear nuevos tipos mediante mecanismos de encapsulamiento y abstracción (TDAs)

## Importancia
- Los tipos proveen contexto implícito para operaciones
- Limitan las operaciones válidas, mejorando la seguridad y legibilidad
- El sistema de tipos de un lenguaje impacta en la calidad y seguridad de los programas

## Relación con el sistema de tipos
- El sistema de tipos organiza y estructura los tipos disponibles, estableciendo reglas para su uso y combinación.
