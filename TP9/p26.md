## Pregunta 26
> Sin usar funciones de alto orden, defina en Haskell las siguientes funciones y en cada caso indique su tipo, si son polimórficas y si son formas funcionales:
> a. suma, suma todos los elementos de una lista de números.
> b. cuadrados, calcula la lista de los cuadrados de los elementos de una lista de números dada.
> c. longitudes, dada una lista de listas, devuelve la lista de sus respectivas longitudes.
> d. ordenados, dada una lista de pares de números, devuelve la lista con aquellos pares en los que la primera componente es menor que el triple de la segunda.
> e. pares, dada una lista de enteros, devuelve una lista con los elementos pares.
> f. masDe, dada una lista de listas y un número, devuelve la lista con aquellas listas cuya longitud es mayor al número dado.

### Respuesta

#### a. suma
- **Tipo:** [Int] -> Int
- **Polimórfica:** No
- **Forma funcional:** No
```haskell
suma [] = 0
suma (x:xs) = x + suma xs
```

#### b. cuadrados
- **Tipo:** [Int] -> [Int]
- **Polimórfica:** No
- **Forma funcional:** No
```haskell
cuadrados [] = []
cuadrados (x:xs) = x*x : cuadrados xs
```

#### c. longitudes
- **Tipo:** [[a]] -> [Int]
- **Polimórfica:** Sí
- **Forma funcional:** No
```haskell
longitudes [] = []
longitudes (x:xs) = length x : longitudes xs
```

#### d. ordenados
- **Tipo:** [(Int, Int)] -> [(Int, Int)]
- **Polimórfica:** No
- **Forma funcional:** No
```haskell
ordenados [] = []
ordenados ((a,b):xs)
  | a < 3*b   = (a,b) : ordenados xs
  | otherwise = ordenados xs
```

#### e. pares
- **Tipo:** [Int] -> [Int]
- **Polimórfica:** No
- **Forma funcional:** No
```haskell
pares [] = []
pares (x:xs)
  | even x    = x : pares xs
  | otherwise = pares xs
```

#### f. masDe
- **Tipo:** [[a]] -> Int -> [[a]]
- **Polimórfica:** Sí
- **Forma funcional:** No
```haskell
masDe [] _ = []
masDe (x:xs) n
  | length x > n = x : masDe xs n
  | otherwise   = masDe xs n
```

> *"Las funciones polimórficas son aquellas cuyo tipo contiene variables de tipo, permitiendo operar sobre diferentes tipos de datos. Una forma funcional es una función que toma otra función como argumento o devuelve una función como resultado."*
> — Slides - Paradigma Funcional - Haskell y SML (p. 18, 35)
