# Resumen Clase 9: Concurrencia

## Introducción
La concurrencia permite que un programa tenga múltiples contextos de ejecución activos, aprovechando recursos para maximizar la velocidad y eficiencia.

## Niveles de abstracción
- **Caja negra:** Concurrencia mediante librerías
- **Intermedio:** Constructores dedicados para tareas y sincronización
- **Experto:** Mecanismos de bajo nivel para concurrencia

## Elementos fundamentales
- Hilos de ejecución
- Estado de ejecución
- Exclusión mutua

## Comunicación y sincronización
- Comunicación por pasaje de mensajes o memoria compartida
- Sincronización cooperativa (espera de tareas) y competitiva (exclusión mutua)
- Comunicación sincrónica y asincrónica

## Importancia
La concurrencia es esencial para aprovechar sistemas multiprocesador y resolver problemas complejos, pero requiere mecanismos adecuados de comunicación y sincronización para evitar errores y garantizar la correctitud.
