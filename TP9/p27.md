## Pregunta 27
> Usando la función de alto orden map, defina las funciones cuadrados y longitudes del ejercicio anterior. En cada caso indique su tipo, si son polimórficas y si son formas funcionales.

### Respuesta

#### a. cuadrados
- **Tipo:** [Int] -> [Int]
- **Polimórfica:** No
- **Forma funcional:** Sí (usa map)
```haskell
cuadrados = map (\x -> x * x)
```

#### b. longitudes
- **Tipo:** [[a]] -> [Int]
- **Polimórfica:** Sí
- **Forma funcional:** Sí (usa map)
```haskell
longitudes = map length
```

> *"Una forma funcional es una función que toma otra función como argumento o devuelve una función como resultado. La función map es un ejemplo clásico de forma funcional en Haskell."*
> — Slides - Paradigma Funcional - Haskell y SML (p. 35)
