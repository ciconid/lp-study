# Resumen Clase 6: Sistema de Tipos

## Introducción
El sistema de tipos es el conjunto de reglas que estructuran y organizan los tipos en un lenguaje de programación, buscando maximizar la seguridad y robustez de los programas.

## Componentes del sistema de tipos
- Definición y asociación de tipos
- Reglas de equivalencia, compatibilidad e inferencia
- Chequeo de tipos (estático o dinámico)
- Polimorfismo (ad-hoc, paramétrico, por inclusión)

## Dureza del tipado
- Grado de restricciones que impone el sistema de tipos
- Un lenguaje fuertemente tipado detecta todos los errores de tipos (type safety)

## Conversión y equivalencia de tipos
- Conversión (coerción) puede ser segura (expansora) o insegura (limitante)
- Equivalencia por nombre, declaración o estructura

## Inferencia de tipos
- El lenguaje puede deducir el tipo de una entidad a partir del contexto

## Importancia
El sistema de tipos influye en la eficiencia, flexibilidad y seguridad del lenguaje. Un buen sistema de tipos previene errores y facilita el desarrollo de software confiable.
