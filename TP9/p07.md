## Pregunta 7
> ¿Qué es una forma funcional o función de alto orden? Ejemplifique.

### Respuesta

Una **forma funcional** o **función de alto orden** (higher-order function) es una función que puede recibir como argumento(s) a otras funciones y/o devolver una función como resultado. Esto permite construir abstracciones poderosas y reutilizables, facilitando la composición y manipulación de funciones.

> *"Funciones de orden superior: Las formas funcionales pueden tomar una función como entrada y retornar una función como resultado. Nos permiten crear funciones más complejas a partir de las funciones primitivas."*
> — Slides - Elementos funcionales en lenguajes de paradigmas convencionales (p. 23)

**Ejemplo en Haskell:**
```haskell
map :: (a -> b) -> [a] -> [b]
map f [] = []
map f (x:xs) = f x : map f xs
```
La función `map` es de alto orden porque toma una función `f` como argumento y la aplica a cada elemento de una lista.

**Ejemplo en Python:**
```python
def aplicar_dos_veces(f, x):
    return f(f(x))

aplicar_dos_veces(lambda x: x + 1, 3)  # Devuelve 5
```
Aquí, `aplicar_dos_veces` es una función de alto orden porque recibe otra función como parámetro.

---

### Información adicional

En la mayoría de los lenguajes funcionales modernos, las funciones de alto orden son fundamentales para expresar patrones de procesamiento sobre colecciones (como `map`, `filter`, `reduce/fold`). Permiten escribir código más declarativo y menos propenso a errores, y son la base de muchas técnicas avanzadas como la composición de funciones y la programación reactiva.
